**Language:** Kotlin

**Description:**
An Android application designed to display jokes and quotes.

**Key Features:**
- **API Integration:** Fetches jokes and quotes from external services (JokeApiService, QuoteApiService).
- **Local Storage:** Uses Room Database for caching or saving data (JokeDAO, UserDAO).
- **User Authentication:** Includes Login and Sign-up functionality.
- **UI:** Features multiple fragments for different views (Home, Saved Jokes, Quotes, etc.).
